HTML UNIT 03 - CSS Selectors 2

You can create multiple class selectors in your CSS code to style different groups of elements. Each class selector should have a unique name to avoid conflicts. You can then apply these classes to different HTML elements as needed. This allows you to reuse styles across multiple elements without having to write the same CSS properties repeatedly.

css selectors are used to "find" (or select) the HTML elements you want to style. In this example, we have created two class selectors: "txt" and "para". The "txt" class selector styles the text color to red, font family to Verdana, font size to 18 pixels, and font weight to 600. The "para" class selector styles the text color to a teal shade, font family to Tahoma, font size to 16 pixels, and font weight to 500.
